What the Sentinel API Does REST · JSON
Your API customers plug Sentinel directly into their existing TMS, load board, or custom system. Their software calls your API — Sentinel runs silently in the background — fraud is caught before it ever reaches a human.
🔍
/scan — FraudShield
Pass an MC or DOT number. Get back a risk score, verdict, and flag breakdown instantly.
📋
/verify — LoadVerify
Pass load details. Sentinel scans all boards and returns a double-booking verdict in real time.
🚛
/carrier — CarrierDNA
Full carrier profile — authority age, insurance, safety rating, history, and community score.
⚡
/webhook — Push Alerts
Sentinel pushes fraud alerts TO their system the moment one is detected. Zero latency.

POST /v1/scan FraudShield — Carrier fraud scan 1 credit
Runs a full fraud check on any carrier MC or DOT number. Returns risk score, verdict, flags, and recommendation.
Request Body
FieldTypeRequiredDescription
mc_numberstringrequiredMC or DOT number
carrier_namestringoptionalCarrier name for cross-reference
phonestringoptionalCarrier phone for VoIP check
addressstringoptionalAddress for mail-drop detection
Response Fields
FieldTypeValues
risk_scoreinteger0–100 (higher = more risk)
verdictstringSAFE / CAUTION / FRAUD
flagsarrayList of detected risk factors
recommendationstringBOOK / VERIFY / DO_NOT_BOOK
credits_usedintegerAlways 1 per scan
example · javascript
const result = await fetch('https://api.sentinelfreightos.com/v1/scan', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    mc_number: 'MC1847291',
    carrier_name: 'Express Freight LLC'
  })
});
const data = await result.json();
// data.verdict === 'FRAUD' → block the load
if (data.verdict === 'FRAUD') blockLoad();
POST /v1/verify LoadVerify — Double-booking detection 1 credit
Scans all major load boards simultaneously for duplicate postings of the same load. Catches double brokering in real time.
request body
{
  "origin": "Atlanta, GA",
  "destination": "Los Angeles, CA",
  "pickup_date": "2026-06-02",
  "pickup_window": "08:00-14:00",
  "weight_lbs": 4000,
  "commodity": "dry_van"
}

// Response
{
  "verdict": "DOUBLE_BOOKED",
  "duplicate_count": 2,
  "found_on": [
    { "board": "DAT", "mc": "MC1847291", "posted": "09:14" },
    { "board": "Truckstop", "mc": "MC2091847", "posted": "09:22" }
  ],
  "action": "DO_NOT_RELEASE"
}

POST /v1/batch/scan Batch — Scan up to 100 carriers at once 1 credit each
Enterprise feature. Pass an array of up to 100 MC numbers. Returns risk scores for all in a single API call. Perfect for nightly carrier audits.
request
{
  "carriers": [
    { "mc_number": "MC1847291" },
    { "mc_number": "MC2204871" },
    { "mc_number": "DOT9912044" }
  ]
}

// Response: array of scan results, same format as /scan
// Ideal for overnight batch audits of your carrier network
